# todo-list-app

## Step to run project
  todo-app-backend: 
  - npm install
  - npm start
  
  todo-list-app: 
  - npm install 
  - npm run dev

## Function
 - Add new todo
 - Display the list of todo
 - Function to edit todo
 - Function to delete todo
 - Function to click on the box to complete the task
 - This project use React, TypeScript, CSS and Node Express.js
